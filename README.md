**Plugin Sederhana** yang memiliki fitur:

- Chat System (Chat Format, Troll, BadWord Filter)
- Player Warp
- Jobs
- Party
- Daily Rewards
- Playtime
- Silent Kick

Plugin ini tidak memiliki konfigurasi sama sekali. Jika ingin mengubah sesuatu, silahkan ubah melalui IDE, lalu build.

### Requirements:

- **Java 21**
- **Paper 1.21.1**
